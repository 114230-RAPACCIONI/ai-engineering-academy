# ADR-004 — IA como colaborador (Mentor)

## Status

Accepted

## Contexto

El objetivo de ZUZU es mejores ingenieros, no generación automática de código.

## Decisión

La IA aumenta la capability humana. Los agents (MVP: **Mentor**) deben explicar, cuestionar, recomendar y revisar — no reemplazar el pensamiento, ocultar el razonamiento ni decidir sin contexto.

## Consecuencias

+ Alineado a la Product Thesis  
− Demos `wow` más lentas que productos de puro codegen
